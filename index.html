Tische verwalten

Auf iPhone oder iPad installieren

  1. Diese Seite in Safari öffnen.
  2. Unten auf das Teilen-Symbol tippen.
  3. „Zum Home-Bildschirm“ auswählen.
  4. Oben rechts auf „Hinzufügen“ tippen.
Für die vollständige Offline-Installation muss die App über eine HTTPS-Adresse oder einen lokalen Webserver geöffnet werden. Beim direkten Öffnen der Datei funktioniert die Kasse, aber die Installation kann eingeschränkt sein.
